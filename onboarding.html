Instalação em 3 passos
1
Instale o Node.js
Necessário para rodar o servidor do DS. Se já tiver instalado, pule para o passo 2.
↗ Baixe em nodejs.org — versão LTS recomendada
2
Configure o acesso ao pacote
Abra o Terminal e cole os dois comandos abaixo.
# Cole no Terminal
echo '@leobertonceli:registry=https://npm.pkg.github.com' >> ~/.npmrc
echo '//npm.pkg.github.com/:_authToken=TOKEN_FORNECIDO_PELO_LEO' >> ~/.npmrc
💡 Isso configura o npm para encontrar o pacote do DS. Só precisa fazer uma vez.
3
Configure o Claude Desktop
Abra o arquivo abaixo no Finder (Cmd + Shift + G) e adicione o bloco JSON.
~/Library/Application Support/Claude/claude_desktop_config.json
{
  "mcpServers": {
    "wonderland-ds": {
      "command": "npx",
      "args": ["-y", "@leobertonceli/wonderland-ds-mcp"]
    }
  }
}
🔄 Após salvar, reinicie o Claude Desktop.
